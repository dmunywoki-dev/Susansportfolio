# Wambui Kariuki Portfolio Website

A beautiful, modern, and fully responsive Vue.js 3 portfolio website built with Vite and Tailwind CSS.

## Features
- Clean, professional design tailored for an academic and policy professional
- Fully responsive (mobile-first)
- Smooth scrolling navigation
- Mobile hamburger menu
- All key sections: Hero, About, Education, Experience, Research, Skills, Contact
- Uses the official professional photo
- Easy to customize and extend

## Quick Start

### 1. Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### 4. Build for production

```bash
npm run build
```

The production-ready files will be in the `dist/` folder.

## Customization

- **Update contact email**: Edit `src/components/Contact.vue`
- **Add more publications**: Extend the Research section
- **Change colors**: Modify `tailwind.config.js` (primary/accent colors)
- **Add new sections**: Create new `.vue` components and import them in `App.vue`

## Image
The professional portrait is located at:
`src/assets/wambui-kariuki.png`

Replace it with an updated high-resolution photo if needed (recommended dimensions ~1200×900px or similar aspect ratio).

## Technologies Used
- Vue 3 (Composition API)
